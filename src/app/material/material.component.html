Angular Material

Introduction

  • Az Angular Material a Material Design egy Angularhoz készült UI könyvtára
    •     ng add @angular/material
  • Elvileg az alábbi lépések történnek meg a telepítéssel
    1. Add project dependencies to package.json
    2. Add the Roboto font to your index.html
    3. Add the Material Design icon font to your index.html
    4. Add a few global CSS styles to
      • Remove margins from body
      • Set height: 100% on html and body
      • Set Roboto as the default application font
  • Nálam ezek valamiért nem történtek meg (leszámítva a package.json dependenciát), ezért az egyik stílust behúztam az angular.json fájlba
    • Ha ez nem történne meg, adjuk hozzá manuálisan
      •     "styles": [
        "node_modules/bootstrap/dist/css/bootstrap.min.css",
        "node_modules/bootstrap-icons/font/bootstrap-icons.css",
        "node_modules/@angular/material/prebuilt-themes/deeppurple-amber.css"
        "src/styles.scss"
    ],
  • A material ikonok megjelenítéséhez szükséges az index.html-ben elhelyezni az alábbi 
        <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
