Bootstrap

  • Reszponzív mobile-first CSS framework
  • Beépített SAS változókkal, grid rendszerrel, mixin-ekkel, komponensekkel rendelkező Javascript plugin
  • Néhány CSS és JS file-ból áll
  • jQuery és properjs függőségei is vannak

Angular integration

  • CDN URL-lel beimportálható
    • CDN
      • A CDN egy földrajzilag elosztott szerverhálózat, amely a weboldalak és alkalmazások statikus tartalmait (például képeket, CSS fájlokat, JavaScript fájlokat, videókat) gyorsítótárazza, és ezekhez a felhasználók számára legközelebbi szerverről biztosít hozzáférést
    • CDN URL
      • A CDN URL egy olyan hivatkozás, amely egy erőforrást (például egy JavaScript könyvtárat, képet vagy stíluslapot) a CDN-en keresztül ér el
    • Előnyei:
      1. Gyors betöltés
        • Mivel a tartalom a felhasználóhoz legközelebbi szerverről érkezik, csökken a válaszidő
      2. Kisebb szerverterhelés
        • Az alkalmazás szervere kevésbé lesz terhelve, mert a statikus tartalmakat a CDN szolgáltatja
      3. Hatékonyabb cache-elés
        • A gyakran használt erőforrások (például jQuery, Bootstrap) már lehet, hogy cache-elve vannak a felhasználó böngészőjében más webhelyek miatt, így nem szükséges újra letölteni őket
  • Egy másik importálási opció, ha felinstalláljuk 
      •   npm install bootstrap
  ng add @ng-bootstrap/ng-bootstrap
  npm i bootstrap-icons
  npm i @types/bootstrap
      • A node_modules mappába telepítődik
      • A min CSS-t a node_modules/bootstrap/dist/css/dist/bootstrap.min.css lokáción található
      • Két módon használhatjuk ezt a CSS fájlt
        1. Az angular.json styles tömbjéhez hozzáadhatjuk (elegánsabb) 
            "styles": [
    "node_modules/bootstrap/dist/css/bootstrap.min.css",
    "node_modules/bootstrap-icons/font/bootstrap-icons.css",
    "src/styles.scss"
  ],
        2. A styles.css fájlba importálás 
            @import "../node_modules/bootstrap/dist/css/dist/bootstrap.min.css";
        3. Valamint ne felejtsük el hozzáadni az alábbi sort az angular.json scripts tömbjéhez
          4.   "scripts": [
    "node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"
  ]

Grid

  • Rugalmas és reszponzív rácsrendszer
  • 12 oszlopos elrendezést használ
  • A grid container-ekből, sorokból (.row), és oszlopokból (.col) áll
  • Az oszlopok egyenletesen oszlanak el, ha nincs megadva szélességük
  • Flexbox alapú, ezért egyszerű az igazítás és a rendezés
  • Egy oszlop tartalmazhat saját .row és .col elemeket, így alrácsokat hozhatunk létre

  indeterminateCheckboxChange(e: any) {
    if (this.state === 'unchecked') {
      this.state = 'indeterminate';
      this.indeterminate.nativeElement.indeterminate = true;
      this.indeterminate.nativeElement.checked = false;
    } else if (this.state === 'indeterminate') {
      this.state = 'checked';
      this.indeterminate.nativeElement.indeterminate = false;
      this.indeterminate.nativeElement.checked = true;
    } else { // this.state === 'checked'
      this.state = 'unchecked';
      this.indeterminate.nativeElement.indeterminate = false;
      this.indeterminate.nativeElement.checked = false;
    }
  }
